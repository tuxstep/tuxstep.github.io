# tuxstep.github.io

Source for [tuxstep.github.io](https://tuxstep.github.io/) — the TuxSTEP project landing page and design document.

GitHub Pages auto-publishes `main` on every push. The current `index.html` is the project's design document — a living spec that evolves alongside project decisions in [tuxstep/tuxstep](https://github.com/tuxstep/tuxstep). Significant design pivots open as PRs here so the rationale gets reviewed and captured.
